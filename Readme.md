# bigwork
**《翻转世界·雷霆战机》**
为Xilinx EGO1开发板设计

一个**竖版打飞机小游戏**，主要功能有：

1.控制飞机移动并判断飞机状态 

2.施放大招:

广域静默: 清除当前屏幕上所有的子弹

3.计算玩家生存的时间并兑换为积分
